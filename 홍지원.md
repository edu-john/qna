# qna
저는 gitbash를 열고 cd~ 를 입력해도 ~(master)가 계속 있는데 왜그럴까요?

작업은 정상적으로 수행되긴 하는데 궁금합니다.

## 질문과 피드백
1. 현재 프로젝트를 `fork` 합니다.
2. fork된 프로젝트를 로컬 컴퓨터로 `clone` 합니다.
3. clone된 프로젝트에 `본인이름.md` 파일명으로 질문과 피드백을 작성합니다.
  ```markdown
  파일명: 강동주.md
  
  # qna
  * git branch에 대해 궁금합니다.
  ```
4. Pull Request를 만들어 보냅니다.
